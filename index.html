FraudZen Hackathon

Join us for an exciting hackathon dedicated to advancing security, privacy, and fraud detection in mobile networks!

Hackathon Overview

The FraudZen Hackathon invites students to tackle real-world challenges in mobile networks using state-of-the-art tools: FraudZen and En-WDM. With these tools, participants will explore and develop innovative approaches in areas like network simulation, privacy protection, and network security strategies.

The hackathon is organized around four main thematic poles, with awards for the best projects in each category.

Hackathon Poles

Pole 1: Mobile Network Architecture, Knowledge Extraction, and Modeling

This pole focuses on the core structure and data dynamics of mobile networks, emphasizing mobility and traffic simulation models. **Objectives** include enhancing simulation precision, extracting meaningful data, and developing new human mobility models or traffic patterns.

Pole 2: Mobile Network Data Privacy Protection through Anonymization

Emphasizing data privacy, this pole introduces students to anonymization techniques like k-anonymity and differential privacy. **Objectives** include protecting privacy in mobile network data, investigating pseudonymized datasets, and balancing privacy with data usability.

Pole 3: Offensive Security

This pole focuses on offensive strategies to simulate fraud tactics in mobile networks. **Objectives** include implementing realistic fraud models, evaluating cost-efficiency for fraudsters, and analyzing detection feasibility.

Pole 4: Defensive Security

Focusing on detecting SIMBox fraud, this pole challenges students to develop AI-based detection techniques. **Objectives** include proposing adaptable fraud detection methods, integrating meta-learners, and establishing effective evaluation metrics.
