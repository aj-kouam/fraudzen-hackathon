About the FraudZen Hackathon

Join us for an innovative hackathon focused on enhancing mobile network security and privacy protection. Participants will work with advanced tools, FraudZen and En-WDM, to tackle real-world challenges.

Mobile Network Architecture

Exploring network architecture and data dynamics to enhance mobility and traffic simulations.

Objectives:
  • Integrate 4G/5G/6G features into FraudZen
  • Propose new mobility models within En-WDM

Privacy Protection through Anonymization

Applying techniques to protect user identities in network data.

Objectives:
  • Implement privacy techniques like k-anonymity
  • Analyze privacy attacks and vulnerabilities

Offensive Security

Simulating fraud tactics to understand and combat SIMBox fraud.

Objectives:
  • Implement fraud models for realistic behaviors
  • Validate cost-efficiency and effectiveness

Defensive Security

Developing advanced AI-based techniques for detecting SIMBox fraud.

Objectives:
  • Propose novel AI-based detection solutions
  • Establish metrics for validating detection effectiveness
